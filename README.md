# E-Commerce Store

A full‑stack Node.js e‑commerce application built with Express and EJS.

---

## 🚀 Features

- Product listing & detail pages  
- Shopping cart with add/remove functionality  
- Server‑side rendering via EJS templates  
- Configurable settings in `config/config.js`  
- Static assets served from `public/` (CSS, JS, images, JSON)

---

## 🛠️ Prerequisites

- [Node.js](https://nodejs.org/) v14 or higher  
- [npm](https://npmjs.com/) (bundled with Node.js)  
- (Optional) [MongoDB](https://www.mongodb.com/) or another database
